# SQL Economic Indicators Analysis (2020–2024)

## Project Overview
This project analyses the growth in key economic indicators between 2020 and 2024 using SQL.

## Objectives
- Measure absolute growth in indicators
- Compare percentage growth across sectors
- Rank indicators based on performance
- Analyse contribution to total growth

## Tools Used
- SQL (MySQL)
- MySQL Workbench

## Key Insights
- GDP is the largest contributor to overall growth
- Financial indicators dominate economic expansion
- Some smaller sectors show high percentage growth
- Growth is highly concentrated among a few indicators
- Social indicators such as education and employment lag behind

## Files in this Project
- queries.sql → contains all SQL queries
- insights.md → contains analysis and findings
- dataset
